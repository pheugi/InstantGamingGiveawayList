## How to Regenerate This README

### Prerequisites

- A POSIX-compatible shell (bash, dash, zsh, etc.)
- [jq](https://stedolan.github.io/jq/) installed and available in PATH

### Quick Usage

Run the generator from the repository root:

```sh
./misc/generateReadme.sh
```

The script reads [json.json](json.json) and overwrites [readme.md](readme.md) only when content changes.

### Configuration (Environment Variables)

| Variable | Default | Description |
| --- | --- | --- |
| `LOCALE` | `fr` | Locale path segment (e.g., `fr`, `en`, `de`) |

Example with custom locale:

```sh
LOCALE=en ./misc/generateReadme.sh
```

### GitHub Actions Automation

This repository includes a workflow ([.github/workflows/generateReadme.yml](.github/workflows/generateReadme.yml)) that automatically regenerates the README when:

- `json.json` is updated
- `misc/generateReadme.sh` is modified
- The workflow file itself changes

You can also trigger it manually via the "Actions" tab → "Generate Optimized README" → "Run workflow".
